# AI-Powered-Exam-Generator 📚

**An intelligent exam generation system that automatically creates multiple-choice questions from various document formats using Transformer-based NLP models.**

# Overview 📘

**This project leverages Google FLAN-T5 to generate structured exam questions from input files such as PDFs, Word documents, PowerPoint presentations, and text files. It processes raw content, cleans and chunks it, and produces high-quality MCQs with adjustable difficulty levels.**

# Features 🚀
**1- AI-based question generation using FLAN-T5**

**2- Multi-format support: .pdf, .docx, .pptx, .txt.**

**3- Difficulty control: Easy, Medium, Hard.**

**4- Robust parsing & retry mechanism for reliable MCQ formatting.**

**5- Smart text chunking for efficient model input handling.**

**6- Customizable number of questions.**

**7- Export the generated exam.**
